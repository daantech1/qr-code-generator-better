# ⚡ QR Code Generator

A sleek, modern QR Code Generator built with pure HTML, CSS and JavaScript.

Simply paste a URL, click **Generate**, preview the QR code instantly, and download it as a high-quality PNG image.

---

## ✨ Features

- 🔗 Convert any URL into a QR Code
- 🖼️ Live QR Code preview
- 📥 Download as PNG
- ⚡ Fast and lightweight
- 🌈 Animated RGB border & glow effects
- 🌌 Modern cyber-style UI
- 📱 Responsive design
- 💾 No backend required
- 🔒 QR Codes never expire (they contain the URL directly)
